# Personal Website

This is a personal portfolio website showcasing my skills, projects, and experience.

## Features

- Responsive design
- Project showcase
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Deployment

This website is deployed on Vercel.

## Local Development

To run this website locally, simply clone the repository and open the `index.html` file in your browser.

```bash
git clone https://github.com/yourusername/personal-website.git
cd personal-website
# Open index.html in your browser
```
